# Pomodachi
A pomodoro timer with a mini game.

Features:
  Breaks: 
    Include a short break that stats after every work session
  
  Themes:
    Has a mint chocolate themed landing page
    And a neopolitan theme.
    
  
  Notification:
    Give you a notification everytime you finish a work session or a break
  
  Stats:
    Has stats about how many work sessions completed,
    Total time spent working,
    And a streak.
